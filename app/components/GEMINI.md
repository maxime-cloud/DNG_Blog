# GEMINI.md — Composants (app/components/)

## Règles de Création
- **Vérification** : NE JAMAIS créer un composant sans vérifier si Nuxt UI possède déjà un équivalent.
- **Customisation** : Toujours modifier les composants de Nuxt UI pour qu'ils correspondent au design du projet.
- **Naming Convention** : Pour les composants personnalisés basés sur Nuxt UI, utiliser le préfixe **"CU"** (Custom User-interface).
  * Exemple : `CUButton` pour `UButton`, `CUInput` pour `UInput`.

## Design Mandates
- Respecter strictement les ombres directionnelles et le style `rounded-none`.
- Utiliser la palette de couleurs définie dans `.claude/rules/design.md`.
