## Prérequis

Assurez-vous d'avoir Node.js (v20+) et `pnpm` installés sur votre machine avant de commencer.

## Cloner le dépôt

```bash
git clone https://github.com/votre-utilisateur/DNGBlog.git
cd DNGBlog
```

## Installer les dépendances

```bash
pnpm install
```

## Variables d'environnement

Copiez le fichier `.env.example` en `.env` et renseignez les valeurs requises : `DATABASE_URL`, `BETTER_AUTH_SECRET`, `BETTER_AUTH_URL`, `PLUNK_SECRET_API_KEY`, `GITHUB_CLIENT_ID` et `GITHUB_CLIENT_SECRET`.

## Lancer le serveur de développement

```bash
pnpm dev
```

Le serveur sera accessible sur `http://localhost:3000`.
